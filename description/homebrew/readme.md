# Элиас homebrew

- **bup**     =>      `brew update && brew upgrade`
- **bout**     =>      `brew outdated`
- **bin**     =>      `brew install`
- **brm**     =>      `brew uninstall`
- **bcl**     =>      `brew cleanup`
- **bls**     =>      `brew list`
- **bsr**     =>      `brew search`
- **binf**     =>      `brew info`
- **bdr**     =>      `brew doctor`
- **bed**     =>      `brew edit`
