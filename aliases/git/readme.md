# Элиас git bash [(back)](../../readme.md)

### BASIC

- **g** => `git`

- **ga** => `git add`

- **gall** => `git add -A`

- **gl** => `git pull`

- **gp** => `git push`

- **gc** => `git commit -m`

- **gcommend** => `git commit --amend --no-edit`

- **gcomedit** => `git commit --ammend -m`

- **gs** => `git status`

- **gss** => `git status -s`

- **gcl** => `git clone`

- **gm** => `git merge`

### STASH

- **gst** => `git stash`

- **gstb** => `git stash branch`

- **gstd** => `git stash drop`

- **gstl** => `git stash list`

- **gstp** => `git stash pop`

### BRANCH

- **gb** => `git branch`

- **gbd** => `git branch -d`

- **gco** => `git checkout`

- **gcob** => `git checkout -b`

- **gcom** => `git checkout master`

- **gcod** => `git checkout develop`

### RESET

- **gsoft** => `git reset --soft HEAD^1`

- **grs** => `git reset --soft`

- **ghard** => `git reset --hard HEAD^1`

- **grh** => `git reset --hard`

### LOG

- **gl** => `git l`

- **glast** => `git l -1 -p --stat`
